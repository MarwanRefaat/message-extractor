# Quick Start Guide

Get your messages extracted in under 2 minutes!

## Install

```bash
./install.sh
```

## Extract

```bash
# Extract everything (iMessage, Gmail, Calendar)
./run.sh --extract-all

# Or just iMessage (works on macOS)
./run.sh --extract-imessage
```
